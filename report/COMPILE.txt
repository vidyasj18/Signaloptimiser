COMPILATION STEPS:
------------------

METHOD 1: Using pdflatex (Most Common)
---------------------------------------
1. Open terminal/command prompt
2. Navigate to report directory:
   cd report

3. Run pdflatex twice (for TOC and references):
   pdflatex main.tex
   pdflatex main.tex

4. Output: main.pdf


METHOD 2: Using Overleaf (Online)
----------------------------------
1. Go to https://www.overleaf.com
2. Create new project > Upload Project
3. Zip the entire "report" folder
4. Upload the zip file
5. Compile automatically (click Recompile)


METHOD 3: Using TeXstudio (Desktop)
------------------------------------
1. Install TeXstudio from texstudio.org
2. Open main.tex
3. Press F5 or click Build & View
4. PDF generates automatically


METHOD 4: Using latexmk (Automatic)
------------------------------------
cd report
latexmk -pdf main.tex

This handles multiple compilations automatically.

ADDING IMAGES:
--------------
1. Place image files in report/images/ folder
   (Supported: .png, .jpg, .pdf, .eps)

2. In any .tex file, add:
   \begin{figure}[h]
       \centering
       \includegraphics[width=0.8\textwidth]{your_image_name}
       \caption{Description of your image}
       \label{fig:your_label}
   \end{figure}

3. Reference in text:
   See Figure \ref{fig:your_label}


COMMON ISSUES & SOLUTIONS:
--------------------------

Issue: "File not found"
Solution: Make sure all .tex files are in sections/ folder

Issue: TOC not updated
Solution: Run pdflatex twice

Issue: Missing packages
Solution: Install texlive-full (Linux) or MiKTeX (Windows)

Issue: Images not showing
Solution: Check images/ folder exists and image names are correct


CLEANING UP:
------------
After compilation, you can delete auxiliary files:

Windows:
del *.aux *.log *.toc *.lof *.lot *.out *.synctex.gz

Linux/Mac:
rm *.aux *.log *.toc *.lof *.lot *.out *.synctex.gz
